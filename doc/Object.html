    promptAndGet(prompt) click to toggle source

    catch和throw： 虽然异常处理机制 raise 和 rescue 是非常适合放弃执行在出问题时，能够跳出一些深层嵌套的结构，再正常处理。 在catch定义一个块的标记给定的名称（这可能是一个符号或一个字符串）。该块正常执行，直到遇到抛出。

    # File Ruby入门/13. 异常处理.rb, line 68
def promptAndGet(prompt)
  print prompt
  res = readline.chomp
  throw :quitRequested if res == "!"
  return res
end
